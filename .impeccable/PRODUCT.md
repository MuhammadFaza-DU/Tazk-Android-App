# Tazk — Product Context

- **Nama App:** Tazk
- **Kategori:** Productivity / Lifestyle (habit-building & gamifikasi)
- **Target Pengguna:** Gen Z (mahasiswa, pekerja muda)
- **Platform:** Mobile — Android saja (min. Android 10 / API 29)
- **Framework:** Flutter, Riverpod, Drift (SQLite), 100% offline
- **Value Utama:** Membantu user mengelola tugas, membangun habit, tetap produktif dan bersemangat lewat gamifikasi RPG-style (XP, level, streak rank, kosmetik).
- **Differentiator:** Task management + habit tracking + gamifikasi RPG dalam satu app, dengan widget homescreen dan Pomodoro/Focus Timer yang terintegrasi ke progress habit.
- **Referensi Visual:** Me+ (Play Store) — hangat, terstruktur, widget-driven.

Detail fitur lengkap: lihat `PRD.md` di root proyek.
